Introduction back to ToC

The purpose of the Building Ontology is to represent the main topological relationships that exists between entities in the building domain. Furthermore, the model aims to represent the main components and systems directly involved in energy related aspects of buildings. This model works as the core module within the BIMERR Ontology Network, where several other domains related to the building industry are connected to it. The model takes as layout the BOT Ontology to represent the topological structure of buildings. Additionally, it reuses concepts from the SAREF4Building Ontology to represent the main components and systems that directly impact the energy consumption of buildings and are suceptible to change in a renovation project.
